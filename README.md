## Instructions
### Requirements
A `C99` compiler

### Build
Files are created in `build`
```sh
make
build/maid test
```

### Installation
Files are placed in `/usr/local`
```sh
sudo make install
sudo ldconfig
```

### Usage
The library can be linked with -lmaid, and a command-line tool `maid` is
available
